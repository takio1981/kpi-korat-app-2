ภาพรวมระบบ

ระบบ KPI 1+11 Agenda KORAT เป็นระบบบันทึกและติดตามตัวชี้วัดสุขภาพ 1+11 ของจังหวัดนครราชสีมา ใช้สำหรับรวบรวมข้อมูลจากหน่วยบริการสาธารณสุขทั้ง 32 อำเภอ แล้วสรุปเป็นรายงานระดับจังหวัด

โครงสร้างข้อมูล

6 ประเด็น (Issues) → 12 ตัวชี้วัดหลัก (Main Indicators)

→ 13 กิจกรรมย่อย (Sub Activities) → 31 รายการข้อมูล (KPI Items)

ปีงบประมาณ

ปีงบประมาณเริ่มตั้งแต่ ตุลาคม ถึง กันยายน ของปีถัดไป (เช่น ปี 2569 = ต.ค. 2568 - ก.ย. 2569)

การเข้าสู่ระบบ

1

เปิดหน้าเว็บระบบ

เข้าที่ URL ของระบบ หรือคลิก "เข้าสู่ระบบ" จากหน้าหลัก

2

กรอก Username และ Password

ใส่ชื่อผู้ใช้งาน (เช่น hosp02536) และรหัสผ่านที่ได้รับจากผู้ดูแลระบบ

3

ระบบจะพาไปยังหน้าที่เหมาะสมตามสิทธิ์

Admin → Admin Dashboard / User → หน้าสรุปรายอำเภอ

หมายเหตุ: หาก Login ผิด 15 ครั้งใน 15 นาที ระบบจะล็อกชั่วคราว

บันทึกข้อมูล KPI

สำหรับ User (หน่วยบริการ) บันทึกข้อมูลตัวชี้วัดของหน่วยบริการตนเอง

1

Login แล้วไปที่เมนู "บันทึกข้อมูล"

ระบบจะแสดงตารางตัวชี้วัดทั้งหมดที่ต้องบันทึก

2

เลือกปีงบประมาณ

เลือกปีงบประมาณที่ต้องการบันทึกจาก dropdown ด้านบน

3

กรอกค่าเป้าหมายและผลงานรายเดือน

กรอกตัวเลขในช่อง "เป้าหมาย" (month=0) และช่องรายเดือน (ต.ค. - ก.ย.) แล้วกด "บันทึก"

4

ตรวจสอบและยืนยัน

ระบบจะแสดงรายการที่เปลี่ยนแปลงให้ตรวจสอบก่อนบันทึก

นำเข้าข้อมูล Excel

1

ไปที่เมนู "นำเข้าข้อมูล"

คลิกเมนู "นำเข้าข้อมูล" จาก Navbar

2

ดาวน์โหลด Template

กดปุ่ม "ดาวน์โหลด Template" เพื่อได้ไฟล์ Excel ตามรูปแบบที่ถูกต้อง

3

กรอกข้อมูลใน Excel แล้วอัปโหลด

กรอกข้อมูลตามรูปแบบ แล้วเลือกไฟล์อัปโหลด ระบบจะแสดง Preview ก่อนนำเข้า

ดูสรุปผลงาน

สำหรับ User ดูสรุปผลงานของหน่วยบริการตนเอง

  • สรุปผลงาน: แสดงเป้าหมาย ผลงาน ร้อยละ ของทุกตัวชี้วัด
  • สรุปรายอำเภอ: เปรียบเทียบข้อมูลระดับอำเภอ

สรุปภาพรวมจังหวัด

หน้าสาธารณะ (ไม่ต้อง Login) แสดงข้อมูลรวมทั้งจังหวัด

  • ดูเป้าหมายรวม ผลงานล่าสุด ร้อยละ ของทุกตัวชี้วัด
  • ดูข้อมูลรายเดือน (ต.ค. - ก.ย.)
  • คลิก เพื่อดูกราฟแนวโน้มรายเดือน
  • Admin สามารถสลับโหมด "หน่วยบริการ / อำเภอ" ได้

รายงาน 1+11 Agenda KORAT

หน้าสาธารณะ แสดงรายงานสรุปตัวชี้วัดตามรูปแบบวาระประชุม

  • เลือกปีงบประมาณ
  • ดูเป้าหมาย ผลงาน ร้อยละ ของทุกตัวชี้วัดหลัก
  • กด "พิมพ์" หรือ "Export PDF" เพื่อนำเสนอ
  • Admin กด "บันทึกผลงานตัวชี้วัดหลัก" เพื่อกรอกข้อมูล

บันทึกผลงานตัวชี้วัดหลัก

บันทึกข้อมูลระดับตัวชี้วัดหลัก (Main Indicator) แยกจากข้อมูลรายหน่วยบริการ ใช้สำหรับข้อมูลระดับจังหวัด/อำเภอ ที่ไม่ได้มาจากหน่วยบริการ

1

ไปที่หน้า "รายงาน 1+11"

คลิกเมนู "รายงาน 1+11" จาก Navbar

2

กดปุ่ม "บันทึกผลงานตัวชี้วัดหลัก"

ปุ่มสีม่วงด้านบน (แสดงเฉพาะ Admin)

3

เลือกระดับ (จังหวัด/อำเภอ) แล้วกดแก้ไข

เลือก "ระดับจังหวัด" หรือเลือกอำเภอจาก dropdown แล้วกดปุ่ม "แก้ไขข้อมูล"

4

กรอกเป้าหมาย/ผลงานรายเดือน แล้วบันทึก

ข้อมูลจะ overlay แทนที่ข้อมูลจาก kpi_records ในรายงาน Agenda

Admin Dashboard

สำหรับ Admin ทุกระดับ ดูภาพรวมการบันทึกข้อมูลทุกหน่วยบริการ

  • สรุปรายหน่วยงาน: ดูจำนวนรายการที่บันทึกแล้ว/ยังไม่บันทึก พร้อม Progress Bar
  • รายงานละเอียด: ดูข้อมูล KPI แต่ละรายการ แยกตามหน่วยบริการ
  • กรองตาม ปีงบ, อำเภอ, ประเด็น, ตัวชี้วัด
  • Export Excel ได้ทั้ง 2 โหมด
  • Drill-down ดูข้อมูลหน่วยบริการเป็นรายหน่วย
Admin CUP จะเห็นเฉพาะหน่วยบริการในอำเภอตนเอง

แก้ไขข้อมูลหน่วยบริการ

Admin แก้ไขข้อมูล KPI ของหน่วยบริการใดก็ได้ ผ่านหน้า "สรุปภาพรวมจังหวัด"

1

ไปที่หน้า "สรุปภาพรวมจังหวัด" แล้วกดปุ่ม "แก้ไขข้อมูลหน่วยบริการ"

2

เลือกหน่วยบริการจากรายชื่อ (กรองอำเภอ/ค้นหาได้)

แสดงเฉพาะหน่วยบริการที่เปิดใช้งาน (status = เปิด)

3

กด "แก้ไขข้อมูล" แล้วแก้ไขค่าในตาราง

ช่องที่เปลี่ยนจะแสดงสีเขียว (เพิ่ม) หรือแดง (ลด)

4

กด "บันทึกข้อมูล" แล้วยืนยัน

ระบบจะแสดงสรุปรายการที่เปลี่ยนแปลงพร้อมคำเตือนถ้ามีค่าลดลง

ส่งออกข้อมูล

สำหรับ Admin ส่งออกข้อมูลไปยังระบบภายนอก

  • ไปที่เมนู "ส่งออกข้อมูล"
  • เลือกปีงบประมาณและดู Preview ข้อมูล
  • กดปุ่ม "ส่งออก" เพื่อ Export ข้อมูล

จัดการระบบ (Super Admin)

เฉพาะ Super Admin เท่านั้น ที่เข้าถึงหน้านี้ได้

หน้านี้มี 3 แท็บ:

โครงสร้าง KPI

เพิ่ม แก้ไข ลบ ประเด็น ตัวชี้วัดหลัก กิจกรรมย่อย และรายการข้อมูล / กำหนดกลุ่มงานให้ตัวชี้วัดหลัก

ผู้ใช้งาน

เพิ่ม แก้ไข ลบ เปิด/ปิดผู้ใช้ กำหนด Role และกลุ่มงาน

Audit Log

ดูประวัติการแก้ไขทั้งหมด (ใคร ทำอะไร เมื่อไหร่ IP อะไร)

จัดการผู้ใช้งาน

Super Admin สามารถจัดการผู้ใช้ได้ในแท็บ "ผู้ใช้งาน" ของหน้า "จัดการระบบ"

เพิ่มผู้ใช้

กรอก Username, Password, ชื่อหน่วยงาน, อำเภอ, Hospcode, Role, กลุ่มงาน

แก้ไขผู้ใช้

กดปุ่ม Edit แล้วแก้ไขข้อมูล (เว้นว่าง password ถ้าไม่เปลี่ยน)

เปิด/ปิดสถานะ

คลิก toggle เพื่อเปิด/ปิดผู้ใช้ (ปิดแล้ว Login ไม่ได้)

ลบผู้ใช้

กดปุ่ม Delete พร้อมยืนยัน (ผู้ใช้ที่มีข้อมูล KPI อาจลบไม่ได้)

ระบบสิทธิ์

สิทธิ์ บันทึก KPI ดูข้อมูล แก้ไขหน่วยบริการ บันทึกตัวชี้วัดหลัก จัดการระบบ

คำถามที่พบบ่อย (FAQ)

Q: ลืมรหัสผ่าน ทำอย่างไร?

A: ติดต่อผู้ดูแลระบบ (Super Admin) เพื่อ Reset รหัสผ่านให้

Q: บันทึกข้อมูลแล้วตัวเลขไม่ขึ้นในภาพรวมจังหวัด?

A: ข้อมูลภาพรวมจังหวัดรวมจากทุกหน่วยบริการ ถ้าบันทึกเฉพาะหน่วยเดียว ต้องรอหน่วยอื่นบันทึกด้วย หรือลอง Refresh หน้า

Q: "ผลงาน (ล่าสุด)" คืออะไร?

A: ระบบจะแสดงค่าจากเดือนล่าสุดที่มีข้อมูล ไม่ใช่ผลรวมทุกเดือน

Q: Admin CUP กับ Admin สสจ. ต่างกันอย่างไร?

A: Admin CUP เห็นทุก KPI แต่เฉพาะอำเภอตัวเอง / Admin สสจ. เห็นทุกอำเภอ แต่เฉพาะ KPI ที่กลุ่มงานตนเองดูแล

Q: Session หมดอายุ ทำอย่างไร?

A: กด "ตกลง" แล้ว Login ใหม่ ระบบจะพากลับไปยังหน้า Login อัตโนมัติ
